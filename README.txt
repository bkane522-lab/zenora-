ZENORA — Correctif audio et boutons

Remplace seulement le fichier script.js de ton dépôt GitHub par celui-ci.

Ce correctif :
- met les vrais MP3 au premier plan ;
- retire les longues pistes génératives qui donnaient un bruit aigu ;
- met Souffle d’hypnose comme son sommeil principal ;
- ajoute des chemins de secours si certains fichiers audio ont été envoyés avec accents ou majuscules ;
- affiche une alerte si un MP3 est introuvable.
